# 凤栖梁

> 古代言情 · 宫廷 · 青梅竹马 · 虐恋情深

---

## 简介

相府二小姐季嫱，因身为皇后的姐姐难产去世，被迫嫁入宫中为后。

她与皇帝胞弟梁王周晔自幼青梅竹马，曾以书信传情，冰凌花为信物。可当周晔从边关凯旋归来，爱人已成皇嫂，咫尺之距，却如天涯之隔。

宫中暗潮汹涌。德妃步步为营，嫣儿伺机而动。季嫱在重重算计中护住孩子、为姐报仇，却不得不将真相深埋心底，任由周晔误解她势利薄情。

一块绣着冰凌花的手帕，能否揭开尘封的真相？

---

## 主要人物

| 人物 | 身份 | 简介 |
|------|------|------|
| **季嫱** | 皇后，相府二小姐 | 冷静隐忍，外柔内刚。被迫入宫为后，与周晔青梅竹马却成叔嫂，一生所求不过是护住孩子、与他重逢 |
| **周晔** | 梁王，皇帝胞弟 | 掌兵权，常驻边关。深情沉稳，心中始终装着季嫱，却因误会与挑拨与她渐行渐远 |
| **德妃** | 皇帝宠妃 | 表面温婉，实则狠辣。先皇后之死另有隐情，她才是幕后黑手 |
| **嫣儿** | 季嫱侍女 → 梁王妃 | 表面忠心，实则野心勃勃。借德妃之局飞上枝头，却不满周晔心中只有皇后 |
