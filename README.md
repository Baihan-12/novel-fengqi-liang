# 凤栖梁

古代言情 · 宫廷 · 在线阅读

> 相府二小姐季嫱因姐姐难产去世被迫入宫为后，与青梅竹马梁王周晔被迫成为叔嫂。冰凌花手帕，能否揭开真相？

## 在线阅读

若已部署 GitHub Pages，访问：`https://你的用户名.github.io/仓库名/`

## 本地预览

用本地服务器打开（因浏览器限制，直接双击 `index.html` 可能无法加载章节）：

```bash
cd 凤栖梁
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 部署到 GitHub Pages

1. 在 GitHub 创建仓库，将本文件夹内容推送上去
2. 仓库设置 → Pages → Source 选择 `main` 分支
3. 保存后等待部署，访问 `https://用户名.github.io/仓库名/`
